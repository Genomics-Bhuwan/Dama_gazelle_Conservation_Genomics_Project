# Dama Gazelle Conservation Genomics Project

## Project Overview
This project focuses on the conservation of the Dama Gazelle (Nanger dama) through genomic studies. By analyzing the genetic diversity, population structure, and evolutionary history of the Dama Gazelle, we aim to inform conservation strategies and improve the management of this endangered species.
